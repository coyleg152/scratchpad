# Scratchpad
An extremely simple app for jotting down notes.

How to run Scratchpad with Linux and GCC:
1) Install raylib on your machine: https://www.raylib.com/index.html
2) Use the Makefile to compile Scratchpad.c
3) Run ./Scratchpad.out

- NOTE: Make sure ModeFont.png is in the same directory as Scratchpad.c,
and compile/run Scratchpad.out in that directory for the font to display.
